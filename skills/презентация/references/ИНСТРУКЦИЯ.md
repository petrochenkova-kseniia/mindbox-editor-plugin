# Скилл /presentation — Генератор презентаций Mindbox

## Что это

Скилл для Claude Code, который создаёт презентации в корпоративном шаблоне Mindbox.
Ты даёшь ему материалы (docx, pdf, текст, расшифровки) — он предлагает структуру,
согласует с тобой, и собирает готовую презентацию.

**Два режима:**
- **Google Slides** — если настроена интеграция. Результат = ссылка на презентацию.
- **PPTX** — если интеграции нет. Результат = файл .pptx, который ты загружаешь в Google Slides сам.

---

## Установка (5 минут)

### Что нужно заранее

- macOS или Linux
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) установлен и работает
- Python 3 (обычно уже есть на Mac)

### Шаги

**1. Скачай папку скилла с Google Drive**

Ссылка: [ВСТАВЬ ССЫЛКУ НА ПАПКУ]

Скачай всю папку к себе. Например, в `~/Downloads/presentation-skill/`.

**2. Запусти установку**

Открой терминал и выполни:

```bash
cd ~/Downloads/presentation-skill
bash setup.sh
```

Скрипт:
- Скопирует файлы скилла в нужные папки
- Установит Python-библиотеки (python-pptx, lxml)
- Проверит, есть ли интеграция с Google Slides

**3. Проверь, что скилл появился**

Открой Claude Code и напиши:
```
/presentation
```

Должен запуститься скилл. Если Claude не видит команду — перезапусти Claude Code.

## Настройка Google Slides (опционально)

Если хочешь, чтобы скилл создавал презентации прямо в Google Slides (а не PPTX-файлы):

**1. Получи файл `oauth_client.json`**

Попроси у владельца скилла (Саши). Это ключ доступа к Google API.

**2. Положи его в нужную папку**

```bash
mkdir -p ~/.claude/google-integration/credentials
cp ~/Downloads/oauth_client.json ~/.claude/google-integration/credentials/
```

**3. Скопируй скрипты авторизации**

```bash
mkdir -p ~/.claude/google-integration/scripts
```

Создай файл `~/.claude/google-integration/scripts/google_auth.py` с содержимым:

```python
#!/usr/bin/env python3
import os
from google_auth_oauthlib.flow import InstalledAppFlow
from google.auth.transport.requests import Request
from google.oauth2.credentials import Credentials

SCOPES = [
    'https://www.googleapis.com/auth/drive',
    'https://www.googleapis.com/auth/documents',
    'https://www.googleapis.com/auth/spreadsheets',
    'https://www.googleapis.com/auth/presentations',
]

BASE_DIR = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
CREDENTIALS_DIR = os.path.join(BASE_DIR, 'credentials')
CLIENT_SECRET = os.path.join(CREDENTIALS_DIR, 'oauth_client.json')
TOKEN_FILE = os.path.join(CREDENTIALS_DIR, 'token.json')

def authenticate():
    creds = None
    if os.path.exists(TOKEN_FILE):
        creds = Credentials.from_authorized_user_file(TOKEN_FILE, SCOPES)
    if not creds or not creds.valid:
        if creds and creds.expired and creds.refresh_token:
            creds.refresh(Request())
        else:
            flow = InstalledAppFlow.from_client_secrets_file(CLIENT_SECRET, SCOPES)
            creds = flow.run_local_server(port=8090)
        with open(TOKEN_FILE, 'w') as f:
            f.write(creds.to_json())
    return creds

def get_credentials():
    if not os.path.exists(TOKEN_FILE):
        raise FileNotFoundError(f"Токен не найден. Запусти: python3 {__file__}")
    creds = Credentials.from_authorized_user_file(TOKEN_FILE, SCOPES)
    if creds and creds.expired and creds.refresh_token:
        creds.refresh(Request())
        with open(TOKEN_FILE, 'w') as f:
            f.write(creds.to_json())
    return creds

if __name__ == '__main__':
    authenticate()
    print("Авторизация успешна!")
```

**4. Установи библиотеки Google**

```bash
pip3 install google-api-python-client google-auth-oauthlib
```

**5. Пройди авторизацию**

```bash
python3 ~/.claude/google-integration/scripts/google_auth.py
```

Откроется браузер → войди под своим Google-аккаунтом → разреши доступ.
Это нужно сделать **один раз**. Токен сохранится и будет обновляться автоматически.

**6. Проверь**

```bash
python3 ~/.claude/presentation-skill/gslides_builder.py check
```

Должно вывести: `Google Slides API: OK`

## Как пользоваться

### Базовый сценарий

1. Открой Claude Code в папке с материалами для презентации
2. Напиши: `/presentation`
3. Приложи файлы: `@кейс.docx @данные.pdf`
4. Claude:
   - Прочитает все файлы
   - Задаст 2–4 вопроса (фокус, аудитория, спикеры, данные)
   - Предложит структуру слайдов с точными текстами
   - Спросит подтверждение
   - Соберёт готовую презентацию

### Что получишь

- **Google Slides режим**: ссылку на презентацию. Открываешь — редактируешь.
- **PPTX режим**: файл .pptx. Загружаешь в Google Slides и редактируешь.

### Что останется доделать вручную

- Вставить скриншоты (Claude оставит пустые слайды с пометками)
- Вставить фото спикеров
- Проверить цифры, если Claude пометил плейсхолдеры `[XX — уточнить]`
- Финальная вычитка текста

## Обновление скилла

Если владелец скилла обновил файлы на Google Drive:

1. Скачай обновлённую папку
2. Запусти `bash setup.sh` ещё раз — он перезапишет файлы

---

## Если что-то не работает

| Проблема | Решение |
|----------|---------|
| `/presentation` не находится | Перезапусти Claude Code |
| `python-pptx not found` | `pip3 install python-pptx lxml` |
| Google Slides: ошибка авторизации | Перезапусти `python3 ~/.claude/google-integration/scripts/google_auth.py` |
| Google Slides: `403 Forbidden` | Попроси владельца добавить твой email в тестовые пользователи OAuth |
| Презентация пустая | Проверь, что `template.pptx` на месте: `ls ~/.claude/presentation-skill/template.pptx` |
