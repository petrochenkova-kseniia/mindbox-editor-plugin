# Установка инструментов

Пошаговая инструкция для установки всех необходимых инструментов. Выполнять по порядку.

**ВАЖНО: НЕ запускать интерактивные команды самостоятельно.** Команды вроде `gh auth login` требуют ввода от пользователя в терминале — Claude не может их выполнить. Вместо этого объяснить пользователю что нужно сделать и попросить выполнить команду самостоятельно в терминале.

## 1. Homebrew

Проверка: `brew --version`

Если не установлен — попросить пользователя выполнить в терминале (Cmd + Пробел → «Terminal» → Enter):
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## 2. GitHub CLI (gh)

Проверка: `gh --version`

Если не установлен (git установится автоматически как зависимость):
```bash
brew install gh
```

## 3. Авторизация в GitHub

Проверка: `gh auth status`

Если не авторизован — **попросить пользователя выполнить самостоятельно в терминале**:
```bash
gh auth login
```

Подсказать что выбрать:
1. GitHub.com
2. **HTTPS** (важно — выбрать именно HTTPS, не SSH)
3. Login with a web browser
4. Скопировать код, открыть ссылку в браузере, вставить код

**НЕ запускать `gh auth login` самостоятельно — команда интерактивная и зависнет.**

Если протокол уже настроен как SSH — переключить на HTTPS:
```bash
gh config set git_protocol https
```

## 4. Настройка git

```bash
gh auth setup-git
git config --global user.name "Имя Фамилия"
git config --global user.email "email@example.com"
```

Спросить у пользователя его имя и email для настройки.
