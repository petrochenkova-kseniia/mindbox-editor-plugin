# Mindbox Editor Plugin

Это плагин Claude Code для команды редакторов Mindbox. Содержит общие редакционные стили, политики и скиллы.

## Критические правила

- **НИКОГДА не пушить напрямую в main** — все изменения только через Pull Request
- **Всегда обновлять версию** в `.claude-plugin/plugin.json` при любых изменениях (без этого обновление не дойдёт до пользователей)
- **Коммиты без Co-Authored-By** — не добавлять строку `Co-Authored-By` в сообщения коммитов

## Внесение изменений

Полный воркфлоу описан в скилле `/mindbox-editor-plugin:mindbox-plugin-contributor`. Использовать его для любых изменений в плагине.
